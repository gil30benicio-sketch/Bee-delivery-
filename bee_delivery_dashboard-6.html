📋 Como inserir dados no Dashboard

1
Clique em 🎯 Definir Metas para configurar a meta mensal de cada mês — o dashboard calcula a meta diária automaticamente.
2
Clique no botão amarelo ＋ Inserir Dados para registrar os pedidos do dia.
3
Selecione o mês e o dia, preencha Produzido e Cancelados e clique Salvar.
4
O dashboard atualiza automaticamente KPIs, gráficos e meta diária.
5
Para ver outro mês, clique no nome do mês na barra de seleção abaixo.
